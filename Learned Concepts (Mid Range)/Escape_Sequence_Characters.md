# Escape Sequence Characters
- There are mainly 4 Escape sequence characters
## 1.New Line
### 📜 \n ---- New Line
## 2.Tab
### 💥 \t ---- Tab
## 3.Vertical Tab
### 🔹 \v ---- Vertical Tab
## 4.Audiable Alert
### 🔸 \a ---- Audiable Alert
- This is not much important
- And we also use this very rarely
> ***Use these Escape Sequence Characters in 'printf' function only***
