# Operators
- 🎛 There are mainly 3 types
 - 1.Unary operators
 - 2.Binary operators
 - 3.Terinary operators
## 1.Unary Operators
- In unary operators there are two types
- a.Post fix
 - Ex: a++  ,means a++ ---- a=a+1
 - Similarly a-- ---- a=a-1
- b.Pre fix
 - Ex: ++a  ,means ++a ---- a=a+1
 - Similarly --a ---- a=a-1
 ### Understand the difference between Post fix and Pre fix unary operators 
 - 1.💥Ex: a=10
  -      b=20
  -      c=(a++)+b
  -   🔚 c=30 only
  - Because,when we use a++ ,it first prints os uses 'a' value only and then increases 'a' to 'a+1'.
- 2.✨Ex: a=10
 -        b=20
 -        c=(++a)+b
 -    🔚 c=31 .
 - Because,when we use ++a ,it first increases 'a' to 'a+1' and then it prints os uses the 'a' value which now storing 'a+1' value.
