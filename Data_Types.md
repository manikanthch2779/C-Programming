# Data Types🔤
🔸 There are mainly 2 types in data  
## 1.Primitive(scalar type)🔢
🔹 Here,there are again 3 types  
---
### A.int
- 🎈Accepts only numbers ***WITHOUT DECIMALS***
 - '%d' ......Used Symbol
 - It takes "2 Bytes" of memory
- 💥There are 3 types of 'int'
#### a.Long int  
#### b.Short int  
#### c.Unsigned int  
---
### B.char  
- 🔥 char means "Alpha Numerical"
 - 🎆 Used Symbols *'%c' ....character or '%s' .....string*
 - 🧨 It takes "1 Byte" of memory
---
### C.float  
- ✨It accepts only numericals ***WITH DECIMALS***
 -'%f' .....Used Symbol
 - It takes "4 Bytes" of memory
- 🌟It again has 2 more types
#### a.double(8 bytes)
#### b.long double(10 bytes)
